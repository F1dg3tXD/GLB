Introduction

Welcome to the GLB Model Viewer, a platform designed to easily view and embed 3D models on the web. This documentation will guide you through uploading, viewing, and embedding your models.

Uploading Models

You can upload your GLB models directly to the platform for viewing.

  1. Click the "Upload Model" button on the main page.
  2. Select your GLB file from your computer.
  3. Once the upload is complete, your model will be displayed in the viewer.

Supported file format: GLB/GLTF

Loading Models via URL

Alternatively, you can load models directly using a URL.

  1. Enter the URL of your GLB model into the URL input field.
  2. Click the "Load Model" button.
  3. Your model will be displayed in the viewer.

Ensure the URL points directly to a GLB file and is publicly accessible.

Using the Model Viewer

The model viewer provides several tools for interacting with your 3D model:

  • Orbit Control: Click and drag to rotate the model.
  • Zoom: Use the scroll wheel to zoom in and out.
  • Pan: Hold the shift key and drag to pan the model.
  • Lights: Enable scene and viewer lighting modes.
  • Animations: Play, pause, and select different animations (if available).
  • Bones: Show or hide the bone structure of the model.

Embedding the Model Viewer

The GLB Model Viewer provides an embed tool to easily integrate the viewer into your own website.

  1. After loading your model (only via URL), click the "Embed" button.
  2. A code snippet will be generated.
  3. Copy the code snippet and paste it into your website's HTML where you want the viewer to appear.

Example Embed Code:

        
          <iframe src="https://f1dg3txd.github.io/GLB/viewer.html?source=your_model_url.glb" width="640" height="480"></iframe>

Customize the width and height attributes to adjust the size of the embedded viewer.

URL Parameters

The following URL parameters are available for customizing the viewer:

Parameter Description Data Type
?source URL of the GLB model to load. string
show_UI Show or hide the user interface. boolean
lights Enable or disable scene lights. boolean
sceneLights Enable or disable scene lights. boolean
animation Index of the animation to play. integer
bones Show or hide bones. boolean

Example URL with URL parameters:

    
      https://f1dg3txd.github.io/GLB/viewer.html?source=your_model_url.glb&show_UI=true&lights=true&sceneLights=false&bones=true&animation=3
